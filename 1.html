Git 명령어 모음

기본 명령어

pwd Print Working directory
ls -la
clear
rm -r direct 하위 디렉토까지 모두 삭제
cat test.txt 터미널에 문서 내용 보여줌

버전 관리 관련 명령어

git add test.txt 작업트리-->스테이지
git commit -m "message" 스테이지-->repository
git commit -am "message2" 작업트리-->스테이지, repository
git init 현재 지역에 지역 저장소 생성
git log commit 기록 살펴보기
git diff 변경 사항 확인

git restore test.txt 수정한 내용을 취소하고 직전 내용으로 돌리기
